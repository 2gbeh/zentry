## Setup

```sh
git clone https://github.com/2gbeh/zentry.git

cd zentry

npm install

# TODO: Update .env

npm run dev
```

| Environment | Web                       | API                           |
| :---------- | :------------------------ | ----------------------------- |
| Local       | http://localhost:3000     | http://localhost:3000/api     |
| Production  | https://zentry.vercel.app | https://zentry.vercel.app/api |
